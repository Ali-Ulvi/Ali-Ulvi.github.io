XML Compare Tool

BY ALI ULVI TALIPOGLU

Sometimes we need to compare XML files ignoring their elements' order but there was not such FREE tool before. Now there is! Download here.
This tool checks two XMLs whether they are same, if not prints the first difference. Great for XML testing. It has options to ignore white spaces, comments and orders.

You need to have Java Runtime Environment installed on your computer.

Source Code: https://github.com/Ali-Ulvi/xmlDiff
